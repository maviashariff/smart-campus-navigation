# Smart Campus Navigation

A full-stack web application for navigating university campus 
using interactive maps and shortest-path routing.

## Tech Stack
React.js | Node.js | Express.js | MongoDB | Leaflet.js

## Features
- Interactive campus map with 17+ locations
- Dijkstra's shortest path algorithm
- Real-time search
- Admin panel with JWT auth
- Fully responsive
- Cloud deployed (Vercel + Render + Atlas)
